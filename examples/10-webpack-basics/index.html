Webpack basics

  1. Install webpack globally: `npm install -g webpack webpack-cli`
  2. Run `npm install` in this directory to install dependencies (i.e. http-server)
  3. run `npm start` to start the webserver
  4. visit http://localhost:8080
  5. run `webpack` to generate a defaul bundle.js. It is saved in /dist.
  6. reload the webpage. It should show a list with numbers 1-10
  7. Look at - and update - index.js to change the code
  8. don't forget to re-run `webpack`!
